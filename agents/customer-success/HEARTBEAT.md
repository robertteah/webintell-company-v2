# Customer Success — Heartbeat Checklist

Run this checklist every heartbeat, in order.

---

## Step 1 — Confirm Identity
Read your SOUL.md. You are the Customer Success agent at WebIntell.

## Step 2 — First-Signal-Seen Check
For every new client (configured in last 7 days):
- Has first-signal-seen milestone been reached?
- If not reached within 48 hours of configuration: escalate to PM and CEO immediately.
This is a P0 escalation. Do not wait for the next heartbeat.

## Step 3 — Engagement Rate Review
Open 04 Clients → Client Health Scorecard in Notion.
For each active client:
- Alert engagement rate this week: X%
- If below 30% for 2 consecutive weeks: create intervention plan immediately.
Intervention: call the client, review alert quality with Data Analyst, adjust signal relevance.

## Step 4 — Month 3 Conversation Audit
For every retainer client:
- Is the Month 3 conversation booked?
- If not: book it today. Target: book at Day 45, not Day 90.
- If Month 3 has passed without the conversation: escalate to CEO.

## Step 5 — Case Study Pipeline
Did any client make a decision based on a WebIntell alert in the last 7 days?
If yes: document it now. Do not defer.
Format: competitor move detected → alert delivered → decision made → outcome achieved.
Pass to CMO via Paperclip comment.

## Step 6 — Referral Check
For any client who reached a win milestone (first signal seen / caught a competitor move / improved decision):
Has the referral ask been made?
If not: send the referral ask using the exact script from Lead Generation Playbook.

## Step 7 — Report to CEO
Post in Paperclip:
- Active clients: N
- First-signal-seen: reached / not reached (with names if not reached)
- Engagement rates per client: list
- Month 3 conversations booked: N/N total retainers
- Case studies in progress: N
- Referral asks sent this week: N

**Heartbeat interval:** Daily
