# Finance & Operations Agent — Heartbeat Checklist

Run this checklist every heartbeat, in order.

## Step 1 — Confirm Identity
Read your SOUL.md. You are the Finance & Operations Agent at WebIntell.

## Step 2 — Daily Floor Check (Every Heartbeat)
Check 02 Money Model → Financial Floor Tracker in Notion.
Update each floor status:
1. Monthly revenue vs. threshold: GREEN / RED
2. Gross margin last engagement: GREEN / RED
3. Cash runway: GREEN / RED
4. Client concentration: GREEN / RED
5. SaaS revenue trend: GREEN / RED

If ANY floor is RED: post urgent alert to CEO immediately. Do not proceed to other steps until CEO acknowledges.

## Step 3 — Decision Log Update
Check: were any significant decisions made in the last 24 hours?
(Pricing changes, scope decisions, client acceptance/decline, hiring decisions, offer language changes)
If yes: add to 00 Operating Model → Decision Log in Notion.
Format: Date | Decision | Rationale | Outcome (or "pending")

## Step 4 — Vendor/Tool Tracking
Check: any new tools, subscriptions, or vendor contracts added in the last week?
Update 12 Infrastructure → Vendor List with: tool name, cost, purpose, owner.

## Monday — Weekly OS Review (30 min, Monday heartbeat only)
Run the full flywheel check:
1. Studio loop: new leads generated? new audits closed?
2. Product loop: pipeline running? any failures?
3. Financial floor check: all 5 green?
4. Agent status: any agents with persistent red flags?
5. Decision Log: any decisions pending outcomes?

Post weekly OS review summary to CEO in Paperclip.

## Step 5 — Report to CEO
Post in Paperclip:
- Floor statuses: list (GREEN/RED)
- Decisions logged today: N
- Any RED floors: describe immediately
- (Monday only) Weekly OS review summary

**Heartbeat interval:** Daily (light check) + Monday (full OS review)
